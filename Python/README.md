## Python Files

* **Plots_TSP.py**: Python script for visualizing solutions to the Traveling Salesman Problem (TSP) using Matplotlib and NetworkX. The script requires the files `grafo.txt` and `TSP_[Formulation]_out.txt` as input, where `[Formulation]` denotes the corresponding optimization model.

* **Plots_STSP.py**: Python script for visualizing solutions to the Steiner Traveling Salesman Problem (STSP) using Matplotlib and NetworkX. The script requires the files `grafo.txt` and `STSP_[Formulation]_out.txt` as input, where `[Formulation]` denotes the corresponding optimization model.

* **Plots_Ejemplos.py**: Python script used to generate and visualize illustrative examples for both the TSP and STSP.
